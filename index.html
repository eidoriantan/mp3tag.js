mp3tag.js

mp3tag.js is an open sourced JavaScript library used to edit the metadata of audio files.
Licensed under the MIT License.

{% octicon mark-github height:20 aria-hidden:true %} Visit GitHub repository

Features

{% octicon code height:72 aria-hidden:true %}

Open Source

It is an open-sourced project and free for everyone to use! You can view the source codes at GitHub

{% octicon browser height:72 aria-hidden:true %}

Node & Browser

It can run on both Node.js and browsers. It was transpiled with Babel to support the older browsers

{% octicon shield-check height:72 aria-hidden:true %}

Standards Compliant

It follows the ID3 Specifications and can validate the user's inputs to comply with the standards

Editor

You can use this editor as a live demonstration. The source codes are available at GitHub. It also works offline so no worries about uploading your files.
